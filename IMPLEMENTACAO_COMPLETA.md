## 🚀 Próximos Passos

1. **Instalar dependências:**
   ```bash
   cd landing-page-upclinic
   npm install
   ```

2. **Testar localmente:**
   ```bash
   npm run dev
   ```

3. **Build:**
   ```bash
   npm run build
   ```

4. **Deploy:**
   ```bash
   firebase deploy --only hosting:site-upclinic
   ```
